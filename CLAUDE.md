# Duflat.com - YouTube Channel Investigator Platform

## Proje Özeti
YouTube video/shorts linki yapıştırıldığında channel'ı bulan, About sayfasındaki sosyal medya linklerini toplayan ve channel arkasındaki ajans/şirketi araştıran profesyonel investigator platformu.

## Repo & Hosting
- **Repo:** https://github.com/martivstaxi/duflat
- **Site:** https://duflat.com
- **Hosting:** GitHub Pages (frontend) + Cloudflare (DNS/SSL)
- **Cloudflare SSL:** Flexible mode

## Planlanan Özellikler
1. **Auth sistemi** - Login/password ile giriş
2. **Search bar** - YouTube video/shorts link yapıştırma
3. **Channel finder** - Video linkinden channel bilgisi çıkarma
4. **Social scraper** - Channel About sayfasından sosyal medya linkleri toplama (Python)
5. **Agency finder** - Channel arkasındaki ajans/şirket araştırma
6. **Contact info** - Ajans iletişim bilgileri gösterme
